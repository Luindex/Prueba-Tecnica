# TaskSync - Prueba Técnica Fullstack Mid React Native

## Descripción
TaskSync es una aplicación móvil desarrollada en **React Native** que permite gestionar tareas de manera eficiente. La app permite **listar, crear, editar, eliminar y marcar tareas como completadas**, incluso en modo offline.  
Esta prueba tiene como objetivo evaluar competencias en **React Native, TypeScript, manejo de estado global, integración de APIs, cache offline, pruebas unitarias y buenas prácticas de desarrollo**.

---

## Requerimientos
- Node.js
- React native y Typescript
- Expo CLI >= 48
- npm 

---

## Instalación y ejecución

1. Clonar el repositorio:

git clone git@github.com:Luindex/Prueba-Tecnica.git
cd tasksync
npm install
